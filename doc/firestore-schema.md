## Collection: `users`

> Created automatically when a user registers. Document ID = Firebase Auth UID.

| Field | Type | Required | Description |
|---|---|---|---|
| `uid` | `string` | ✅ | Firebase Auth UID (same as document ID) |
| `email` | `string` | ✅ | User's email address |
| `displayName` | `string \| null` | ✅ | Full name or display name |
| `photoURL` | `string \| null` | ❌ | Profile photo URL (Google avatar or uploaded) |
| `role` | `"student" \| "admin"` | ✅ | Access level. Default: `"student"` |
| `createdAt` | `Timestamp` | ✅ | Firestore server timestamp when the account was created |

### Example Document
```json
{
  "uid": "SQkDQOvPeLgolGlclGIviY7HOH22",
  "email": "student@example.com",
  "displayName": "Ahmad Ghawi",
  "photoURL": null,
  "role": "student",
  "createdAt": "2026-04-28T09:48:46.488Z"
}
```

### Notes
- **Never store passwords** in Firestore. Firebase Auth handles credentials.
- To make someone an admin: update `role` to `"admin"` in Firestore console.
- The `role` field is read by `AuthContext.tsx` after every login to determine routing.

## Collection: `courses`

> Managed exclusively by admins via the Admin panel. Document ID is auto-generated by Firestore.

| Field | Type | Required | Description |
|---|---|---|---|
| `id` | `string` | ✅ | Auto-generated Firestore document ID |
| `title` | `string` | ✅ | Course title (e.g., "JavaScript for Beginners") |
| `description` | `string` | ✅ | Short course summary (shown on catalog card) |
| `longDescription` | `string` | ❌ | Full rich-text description for the course detail page |
| `category` | `string` | ✅ | Category tag (e.g., `"JavaScript"`, `"Python"`, `"DevOps"`) |
| `level` | `"Beginner" \| "Intermediate" \| "Advanced"` | ✅ | Difficulty level |
| `thumbnailUrl` | `string` | ✅ | URL of the course cover image |
| `instructorName` | `string` | ✅ | Name of the instructor |
| `instructorId` | `string` | ❌ | UID of the instructor (if they have an account) |
| `price` | `number` | ✅ | Price in USD. Use `0` for free courses |
| `isFree` | `boolean` | ✅ | `true` if the course is free. Derived from `price === 0` |
| `isPublished` | `boolean` | ✅ | `true` = visible on the public catalog. `false` = draft |
| `rating` | `number` | ❌ | Average rating (0–5). Updated by the system |
| `totalStudents` | `number` | ❌ | Total enrolled students. Updated by the system |
| `totalDuration` | `number` | ❌ | Total course duration in seconds. Auto-calculated |
| `tags` | `string[]` | ❌ | Array of searchable tags (e.g., `["react", "hooks", "frontend"]`) |
| `modules` | `Module[]` | ✅ | Ordered array of course modules (see structure below) |
| `createdAt` | `Timestamp` | ✅ | When the course was created |
| `updatedAt` | `Timestamp` | ✅ | When the course was last updated |

### Module Structure (nested inside `courses.modules`)

| Field | Type | Required | Description |
|---|---|---|---|
| `id` | `string` | ✅ | Unique ID for the module (use `crypto.randomUUID()`) |
| `title` | `string` | ✅ | Module title (e.g., "Module 1: Getting Started") |
| `order` | `number` | ✅ | Display order (0-indexed) |
| `lessons` | `Lesson[]` | ✅ | Ordered array of lessons in this module |

### Lesson Structure (nested inside `modules.lessons`)

| Field | Type | Required | Description |
|---|---|---|---|
| `id` | `string` | ✅ | Unique ID for the lesson (use `crypto.randomUUID()`) |
| `title` | `string` | ✅ | Lesson title (e.g., "Variables and Data Types") |
| `order` | `number` | ✅ | Display order within the module (0-indexed) |
| `videoUrl` | `string` | ✅ | **YouTube embed URL** (see format below) |
| `duration` | `number` | ✅ | Lesson duration in **seconds** (e.g., 600 = 10 min) |
| `isFree` | `boolean` | ❌ | `true` = this lesson is a free preview. Default: `false` |

## Collection: `enrollments`

> Created when a student clicks "Enroll" on a course. Document ID is auto-generated.

| Field | Type | Required | Description |
|---|---|---|---|
| `id` | `string` | ✅ | Auto-generated Firestore document ID |
| `userId` | `string` | ✅ | UID of the enrolled student (FK → `users`) |
| `courseId` | `string` | ✅ | ID of the course (FK → `courses`) |
| `progress` | `number` | ✅ | Overall completion percentage (0–100). Auto-calculated |
| `completedLessons` | `string[]` | ✅ | Array of `lessonId` strings the student has completed |
| `lastLessonId` | `string \| null` | ❌ | The last lesson the student watched (for "continue" button) |
| `lastModuleId` | `string \| null` | ❌ | The last module the student was in |
| `enrolledAt` | `Timestamp` | ✅ | When the student enrolled |
| `completedAt` | `Timestamp \| null` | ❌ | When the student completed the course (if `progress === 100`) |

### Example Document
```json
{
  "id": "enroll-xyz",
  "userId": "SQkDQOvPeLgolGlclGIviY7HOH22",
  "courseId": "abc123",
  "progress": 25,
  "completedLessons": ["les-1"],
  "lastLessonId": "les-1",
  "lastModuleId": "mod-1",
  "enrolledAt": "2026-04-29T11:00:00Z",
  "completedAt": null
}
```

### Notes
- **Query a student's enrollments:** `where("userId", "==", currentUser.uid)`
- **Check if enrolled:** query `enrollments` where `userId == currentUser.uid AND courseId == targetCourseId`
- **Mark lesson complete:** add `lessonId` to `completedLessons` array, recalculate `progress`

## Security Rules Summary

Deploy these in the Firebase Console → Firestore → Rules:

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    function isAdmin() {
      return request.auth != null &&
        get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == "admin";
    }

    // Users: own doc or admin
    match /users/{userId} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
      allow read: if isAdmin();
    }

    // Courses: public read, admin write
    match /courses/{courseId} {
      allow read: if true;
      allow write: if isAdmin();
    }

    // Enrollments: own records only, admin can read all
    match /enrollments/{enrollmentId} {
      allow read, write: if request.auth != null &&
        request.auth.uid == resource.data.userId;
      allow read: if isAdmin();
      // Allow creating new enrollment (resource.data doesn't exist yet)
      allow create: if request.auth != null &&
        request.auth.uid == request.resource.data.userId;
    }
  }
}
```

## How to Query — Code Snippets

### Get all published courses (public catalog)
```typescript
import { collection, query, where, getDocs, orderBy } from "firebase/firestore";
import { db } from "@/lib/firebase";

const q = query(
  collection(db, "courses"),
  where("isPublished", "==", true),
  orderBy("createdAt", "desc")
);
const snapshot = await getDocs(q);
const courses = snapshot.docs.map(doc => ({ id: doc.id, ...doc.data() }));
```

### Get a single course by ID
```typescript
import { doc, getDoc } from "firebase/firestore";

const ref = doc(db, "courses", courseId);
const snap = await getDoc(ref);
if (snap.exists()) {
  const course = { id: snap.id, ...snap.data() };
}
```

### Get enrollments for current student
```typescript
import { collection, query, where, getDocs } from "firebase/firestore";

const q = query(
  collection(db, "enrollments"),
  where("userId", "==", currentUser.uid)
);
const snapshot = await getDocs(q);
const enrollments = snapshot.docs.map(doc => ({ id: doc.id, ...doc.data() }));
```

### Mark a lesson as complete
```typescript
import { doc, updateDoc, arrayUnion } from "firebase/firestore";

const enrollmentRef = doc(db, "enrollments", enrollmentId);
await updateDoc(enrollmentRef, {
  completedLessons: arrayUnion(lessonId),
  lastLessonId: lessonId,
  progress: newProgressPercentage, // recalculate: (completedLessons.length / totalLessons) * 100
});
```

### Get a user's profile
```typescript
import { doc, getDoc } from "firebase/firestore";

const ref = doc(db, "users", uid);
const snap = await getDoc(ref);
const user = snap.exists() ? { id: snap.id, ...snap.data() } : null;
```

## YouTube Video URL Format

> ⚠️ **Important:** Always store the **embed URL**, not the regular watch URL.

| Type | Format | Example |
|---|---|---|
| ✅ **Store this** (embed) | `https://www.youtube.com/embed/{VIDEO_ID}` | `https://www.youtube.com/embed/W6NZfCO5SIk` |
| ❌ **Not this** (watch) | `https://www.youtube.com/watch?v={VIDEO_ID}` | `https://www.youtube.com/watch?v=W6NZfCO5SIk` |
| ❌ **Not this** (short) | `https://youtu.be/{VIDEO_ID}` | `https://youtu.be/W6NZfCO5SIk` |

### How to get the Video ID
From a YouTube URL like `https://www.youtube.com/watch?v=W6NZfCO5SIk`:
- The Video ID is the part after `v=` → `W6NZfCO5SIk`
- Embed URL becomes: `https://www.youtube.com/embed/W6NZfCO5SIk`

### Embedding in the video player (`/learn` page)
```tsx
<iframe
  src={lesson.videoUrl}
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowFullScreen
  className="w-full aspect-video rounded-2xl"
/>
```

## TypeScript Interfaces

Copy these into `src/types/index.ts` for type safety across the project:

```typescript
import { Timestamp } from "firebase/firestore";

export interface User {
  uid: string;
  email: string | null;
  displayName: string | null;
  photoURL?: string | null;
  role: "student" | "admin";
  createdAt: Timestamp;
}

export interface Lesson {
  id: string;
  title: string;
  order: number;
  videoUrl: string;       // YouTube embed URL
  duration: number;       // seconds
  isFree?: boolean;
  description?: string;
}

export interface Module {
  id: string;
  title: string;
  order: number;
  lessons: Lesson[];
}

export interface Course {
  id: string;
  title: string;
  description: string;
  longDescription?: string;
  category: string;
  level: "Beginner" | "Intermediate" | "Advanced";
  thumbnailUrl: string;
  instructorName: string;
  instructorId?: string;
  price: number;
  isFree: boolean;
  isPublished: boolean;
  rating?: number;
  totalStudents?: number;
  totalDuration?: number;
  tags?: string[];
  modules: Module[];
  createdAt: Timestamp;
  updatedAt: Timestamp;
}

export interface Enrollment {
  id: string;
  userId: string;
  courseId: string;
  progress: number;             // 0–100
  completedLessons: string[];   // lessonIds
  lastLessonId?: string | null;
  lastModuleId?: string | null;
  enrolledAt: Timestamp;
  completedAt?: Timestamp | null;
}
```
